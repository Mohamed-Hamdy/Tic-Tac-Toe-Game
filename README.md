# Tic-Tac-Toe-Game
Tic Tac Toe Game is application that have three type of playing in one application 
* Play against computer 
* play with your friend 
* play online with friend


# Requirement
* Install Javafx Library
* java 8 JDK
* Netbeans IDE or any java IDE
